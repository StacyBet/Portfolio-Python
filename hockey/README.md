## Проект: "Хоккей: расчет числа Лемтюгова"

В моем распоряжении данные об игроках команд КХЛ, о трансферах игроков по командам, личные данные игроков и статистика их достижений. Данные актуальны на июнь 2023г.

### Цель исследования:

Рассчитать число Лемтюгова - аналог числа Эрдёша-Бэйкона. С помощью данного числа можно измерять "близость" хоккеистов к Николаю Лемтюгову через команды, в которых они играли.

### Ход исследования:

Данные я получу из 5ти файлов:

players_data.csv - датасет из всех игроков, заигранных за команды КХЛ, начиная с сезона 2008-2009. Данные с сайта eliteprospects.com.
transfers.csv - датасет всех трансферов игроков из всех лиг, первый трансфер 14/12/2007. Данные с сайта eliteprospects.com.
players_personal_data.csv - датасет с личными данными игроков.
player_stats.csv - датасет со статистикой игроков.
players.csv - датасет с данными о времени игры в командах.
Все данные "сырые", поэтому, изначально необходимо изучить их, провести предобработку для лучшей дальнейшей работы, а также правильности выводов.

### Таким образом, мое исследование пройдет в 6 этапов:

1. Изучение общей информации о датасетах.
2. Предобработка данных.
3. Анализ данных.
4. Расчет числа Лемтюгова.
5. Проверка гипотез.
6. Вывод.
